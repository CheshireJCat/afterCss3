# After Web Platform Promo

A Remotion promotional video for:

- https://cheshirejcat.github.io/afterCss3/
- https://cheshirejcat.github.io/afterHtml5/

The video highlights both projects as practical, bilingual, demo-driven maps of modern web platform capabilities after the CSS3 and HTML5 branding eras.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run poster
npm run render
```

Rendered outputs are written to `out/`, which is intentionally ignored by Git.
