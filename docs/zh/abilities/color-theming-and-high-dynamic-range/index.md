# 颜色、主题与高动态范围

这个分类包含 15 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [现代 rgb/hsl 语法](modern-rgb-hsl-syntax.md) - 空格分隔、斜杠 alpha，统一颜色函数语法。
- [HWB](hwb.md) - 按 hue/white/black 表达颜色。
- [Lab/LCH/OKLab/OKLCH](lab-lch-oklab-oklch.md) - 感知均匀颜色空间，适合设计系统和渐变。
- [wide-gamut color()](wide-gamut-color.md) - Display-P3、Rec.2020、A98、ProPhoto、XYZ 等广色域。
- [color-mix()](color-mix.md) - 在指定颜色空间混色。
- [相对颜色语法](relative-color-syntax.md) - 从已有颜色提取通道并派生新颜色。
- [light-dark()](light-dark.md) - 与 color-scheme 协作的明暗主题值选择。
- [contrast-color()](contrast-color.md) - 根据背景选择可读前景色。
- [color-contrast()](color-contrast.md) - 从候选颜色中选满足对比度的颜色。
- [颜色插值方法](color-interpolation-methods.md) - 控制渐变和动画使用的颜色空间和 hue 插值。
- [color-scheme](color-scheme.md) - 让 UA 控件、滚动条、表单跟随主题。
- [forced colors](forced-colors.md) - 高对比/强制颜色模式适配。
- [accent-color](accent-color.md) - 原生 checkbox/radio/range/progress 等强调色。
- [print-color-adjust](print-color-adjust.md) - 控制打印或节省墨水时的颜色调整。
- [HDR color](hdr-color.md) - 面向高动态范围显示的颜色输出控制。
