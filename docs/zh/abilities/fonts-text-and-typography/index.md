# 字体、文字与排版

这个分类包含 18 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [Variable fonts](variable-fonts.md) - 可变轴、光学尺寸、字体技术声明。
- [字体加载控制](font-loading-control.md) - 控制 web font 加载时机、fallback 和动态加载。
- [字体指标覆盖](font-metric-overrides.md) - 减少字体替换导致的 CLS 和排版跳动。
- [字体调色板](font-palettes.md) - 彩色字体调色和主题化。
- [OpenType 控制](opentype-controls.md) - 连字、数字、东亚排版、特性开关。
- [字体合成控制](font-synthesis-control.md) - 控制浏览器是否合成粗体、斜体、小型大写等。
- [text-wrap](text-wrap.md) - 标题平衡换行、正文更优换行。
- [white-space 分解](white-space-decomposition.md) - 把 white-space 的折叠与换行行为拆开控制。
- [断词与连字符增强](word-breaking-and-hyphenation-enhancements.md) - 更细控制断词、连字符和自然语言换行。
- [文本装饰增强](text-decoration-enhancements.md) - 下划线厚度、偏移、跳墨和字体推荐位置。
- [强调与特殊装饰](emphasis-and-special-decorations.md) - 东亚着重号、拼写/语法错误装饰。
- [text-box trimming](text-box-trimming.md) - 去除字体盒多余 leading，实现更精确垂直对齐。
- [首字下沉](drop-caps.md) - 报刊式首字下沉和首字对齐。
- [基线与行网格](baselines-and-line-grids.md) - 跨脚本基线、节奏尺寸、行网格。
- [间距与自动排版](spacing-and-automatic-typography.md) - 中日韩标点、拉丁/东亚混排空隙和字间处理。
- [Ruby annotation](ruby-annotation.md) - 日文/中文拼音、注音、旁注布局。
- [text-indent 扩展](text-indent-extensions.md) - 悬挂缩进和每行缩进。
- [line-clamp](line-clamp.md) - 多行截断和块省略。
