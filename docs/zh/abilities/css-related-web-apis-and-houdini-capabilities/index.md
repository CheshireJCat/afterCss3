# CSS 相关 Web APIs 与 Houdini 能力

这个分类包含 11 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [CSSOM](cssom.md) - 读取、解析、序列化和操作 CSS 规则。
- [CSSOM View](cssom-view.md) - 视口、滚动、布局盒查询。
- [Geometry interfaces](geometry-interfaces.md) - 几何对象、矩阵、坐标转换。
- [Resize Observer](resize-observer.md) - 观察元素尺寸变化，补足纯 CSS 无法通知 JS 的场景。
- [Font Loading API](font-loading-api.md) - 脚本控制字体加载和字体集合状态。
- [Web Animations API](web-animations-api.md) - 与 CSS 动画模型共享 timing/effect 的 JS API。
- [Custom Highlight API](custom-highlight-api.md) - 用 JS 注册范围，CSS 用 ::highlight(name) 样式化。
- [Properties and Values API](properties-and-values-api.md) - 运行时注册可类型化、可动画的 CSS 自定义属性。
- [Typed OM](typed-om.md) - 以 typed object 表示 CSS 值。
- [Paint/Layout/Animation Worklets](paint-layout-animation-worklets.md) - 浏览器渲染管线扩展点。
- [View Transition API](view-transition-api.md) - 与 CSS view-transition-* 伪元素协作。
