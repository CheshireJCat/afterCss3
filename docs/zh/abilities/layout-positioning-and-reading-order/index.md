# 布局、定位与阅读顺序

这个分类包含 16 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [Grid layout](grid-layout.md) - 二维布局、命名区域、自动放置。
- [Subgrid](subgrid.md) - 子网格继承父网格轨道，对齐深层内容。
- [Masonry / Grid Level 3](masonry-grid-level-3.md) - 瀑布流类布局进入标准草案。
- [Flexbox](flexbox.md) - 一维弹性布局和空间分配。
- [Box Alignment](box-alignment.md) - 跨 flex/grid/block 的统一对齐模型。
- [多关键字 display](multi-keyword-display.md) - 拆分 outer/inner display，表达更精确。
- [Flow root](flow-root.md) - 无需 clearfix 创建新的块格式化上下文。
- [Logical properties](logical-properties.md) - 按书写方向映射布局、边距、边框和尺寸。
- [Writing modes](writing-modes.md) - 竖排、横竖混排、国际化排版。
- [Containment](containment.md) - 隔离布局/样式/绘制，提高性能并支持容器查询。
- [Anchor positioning](anchor-positioning.md) - 无 JS 定位 tooltip、popover、菜单等依附浮层。
- [Fallback positioning](fallback-positioning.md) - 浮层碰撞时自动翻转/改位。
- [Reading flow/order](reading-flow-order.md) - 让辅助技术和 Tab 顺序跟随视觉布局。
- [Shapes and exclusions](shapes-and-exclusions.md) - 非矩形文字环绕、复杂浮动区域。
- [Multicol and fragmentation](multicol-and-fragmentation.md) - 多列排版和跨页/跨列断点控制。
- [Ruby layout](ruby-layout.md) - 东亚注音/旁注排版。
