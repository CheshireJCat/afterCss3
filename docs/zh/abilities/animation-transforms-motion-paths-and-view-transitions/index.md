# 动画、变换、运动路径与视图过渡

这个分类包含 14 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [Transitions / Animations](transitions-animations.md) - CSS 属性随时间变化。
- [Individual transforms](individual-transforms.md) - 单独控制变换组件，组合动画更稳定。
- [3D transforms](3d-transforms.md) - 三维变换和透视。
- [Motion path](motion-path.md) - 沿路径定位和动画。
- [Easing functions](easing-functions.md) - 分段线性、贝塞尔、步进缓动。
- [Discrete transitions](discrete-transitions.md) - display、overlay 等离散值也可参与过渡。
- [@starting-style](at-starting-style.md) - 进入 DOM 或从 display:none 出现时有起始样式。
- [animation-composition](animation-composition.md) - 多个动画之间的合成模式。
- [Scroll-driven animations](scroll-driven-animations.md) - 滚动进度或元素进入视口驱动动画。
- [animation-range](animation-range.md) - 限定 scroll/view timeline 的有效区间。
- [View transitions](view-transitions.md) - 页面状态或跨文档导航过渡。
- [View transition classes/types](view-transition-classes-types.md) - 复用过渡样式、跨页面过渡声明。
- [Animation triggers](animation-triggers.md) - 用状态/事件触发动画，而非只靠时间轴。
- [Pointer-driven animations](pointer-driven-animations.md) - 鼠标/指针位置驱动动画。
