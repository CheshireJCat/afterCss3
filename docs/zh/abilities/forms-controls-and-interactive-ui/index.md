# 表单、控件与交互 UI

这个分类包含 12 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [accent-color](accent-color.md) - 快速定制原生表单强调色。
- [appearance/base appearance](appearance-base-appearance.md) - 控制原生控件外观和可定制 select 模式。
- [Customizable select](customizable-select.md) - 原生 select 的按钮、picker、option、图标可样式化。
- [Form control parts](form-control-parts.md) - 文件按钮、下拉箭头、勾选标记、滑块轨道等部件。
- [caret-color](caret-color.md) - 输入光标颜色控制。
- [user-select](user-select.md) - 控制文本选择行为。
- [resize logical values](resize-logical-values.md) - 按逻辑轴调整可缩放控件。
- [cursor url-set](cursor-url-set.md) - 更清晰地提供多分辨率指针资源。
- [outline-color:auto](outline-color-auto.md) - 让 UA 选择可访问的焦点轮廓颜色。
- [highlight pseudos](highlight-pseudos.md) - 选区、文本片段和自定义范围高亮。
- [color-scheme + forced-colors](color-scheme-plus-forced-colors.md) - 让原生控件和高对比模式协作。
- [reading-flow/order](reading-flow-order.md) - 视觉重排场景下控制可访问阅读与 Tab 顺序。
