# 尺寸、单位、数学和值函数

这个分类包含 17 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [Intrinsic sizing](intrinsic-sizing.md) - 基于内容自然尺寸和约束计算宽高。
- [aspect-ratio](aspect-ratio.md) - 用比例参与布局，不必 padding hack。
- [比较函数](comparison-functions.md) - 流式排版、响应式尺寸、边界控制。
- [进阶数学函数](advanced-math-functions.md) - 原生网格计算、三角布局、指数/取整/符号判断。
- [数学常量](math-constants.md) - 在 calc 表达式中表示标准数学常量。
- [动态视口单位](dynamic-viewport-units.md) - 解决移动端地址栏变化导致的 100vh 问题。
- [逻辑视口单位](logical-viewport-units.md) - 按 inline/block 轴定义视口单位。
- [字体相关单位](font-relative-units.md) - 用字帽高、汉字水平方块、行高、根字体指标参与布局。
- [interpolate-size](interpolate-size.md) - 允许 auto、min-content 等 intrinsic size 参与动画插值。
- [calc-size()](calc-size.md) - 对 intrinsic size 做计算并参与动画。
- [typed attr()](typed-attr.md) - 从 HTML 属性读取 typed value，用于 content 以外属性。
- [树计数函数](tree-counting-functions.md) - 按兄弟序号/总数做 stagger、布局公式、颜色分布。
- [随机与选择函数](random-and-selection-functions.md) - 原生随机数或随机项，避免预生成 class。
- [fallback/条件值函数](fallback-and-conditional-value-functions.md) - 值级 fallback、状态切换、进度归一化、数值混合。
- [URL 请求修饰](url-request-modifiers.md) - 为 CSS 资源请求表达完整性等元信息。
- [anchor-size()](anchor-size.md) - 取被锚定元素尺寸参与布局。
- [field-sizing](field-sizing.md) - input/textarea 根据内容自适应尺寸。
