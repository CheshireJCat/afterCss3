# 分页、打印与出版排版

这个分类包含 9 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [@page](at-page.md) - 分页媒介尺寸、边距和页面级样式。
- [page selectors](page-selectors.md) - 首页、左右页、空白页分别控制。
- [margin boxes](margin-boxes.md) - 页眉页脚和边距区域内容。
- [named pages](named-pages.md) - 不同章节使用不同页面模板。
- [bleed and marks](bleed-and-marks.md) - 印刷出血和裁切/套准标记。
- [fragmentation](fragmentation.md) - 跨页、跨列、跨区域断点控制。
- [page floats](page-floats.md) - 出版物中的图表浮动与延后放置。
- [footnotes](footnotes.md) - 脚注区域、脚注编号和脚注规则。
- [CSS Speech](css-speech.md) - 面向语音渲染的样式控制。
