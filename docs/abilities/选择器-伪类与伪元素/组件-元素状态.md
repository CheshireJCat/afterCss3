## 简明说明

自定义元素定义状态、方向性、链接状态等。

## 代表语法

```css
:defined, :dir(), :any-link, :local-link
```

## 什么时候用

- 当你需要解决「自定义元素定义状态、方向性、链接状态等。」这类问题时，可以优先考虑它。
- 如果状态里包含“草案”“支持有限”“实现推进中”，建议用 `@supports`、特性检测或保守 fallback。
- 如果目标用户浏览器较旧，先从 MDN 与 Can I Use 核对支持矩阵，再决定是否进入生产。
