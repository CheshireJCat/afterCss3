# 选择器、伪类与伪元素

这个分类包含 15 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [函数式选择器](函数式选择器.md) — 组合复杂条件、降低或控制 specificity、父/关系选择。
- [相对选择器](相对选择器.md) — 基于子元素、后续兄弟、关系链选择当前元素。
- [增强 nth-child](增强-nth-child.md) — 按子集计算结构位置。
- [焦点与可访问状态](焦点与可访问状态.md) — 键盘焦点可视化、父容器焦点联动。
- [组件/元素状态](组件-元素状态.md) — 自定义元素定义状态、方向性、链接状态等。
- [表单状态](表单状态.md) — 用 CSS 响应输入合法性和默认/必填/用户交互状态。
- [弹层与顶层状态](弹层与顶层状态.md) — 选择 details/dialog/popover/fullscreen 等开放或顶层状态。
- [Shadow DOM 选择](shadow-dom-选择.md) — Web Components 内外部样式边界和可暴露部件。
- [自定义状态](自定义状态.md) — 自定义元素通过 ElementInternals 暴露状态给 CSS。
- [视图过渡状态](视图过渡状态.md) — 在 View Transition 期间选择根或过渡类型。
- [列选择器](列选择器.md) — 针对表格列关系和列序号选择。
- [现代伪元素](现代伪元素.md) — 列表标记、占位符、文件按钮、选区、高亮、顶层背景等可样式化。
- [View Transition 伪元素](view-transition-伪元素.md) — 定义页面状态变化的捕获图层和动画。
- [原生滚动控件伪元素](原生滚动控件伪元素.md) — CSS-only 轮播/分页指示与滚动按钮。
- [原生表单部件伪元素](原生表单部件伪元素.md) — 更细粒度地样式化 select、checkbox、range、date/time 等控件。
