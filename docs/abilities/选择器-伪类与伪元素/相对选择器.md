# 相对选择器

<FeatureLinks mdn="https://developer.mozilla.org/en-US/search?q=%3Ahas%28%3E%20CSS" caniuse="https://caniuse.com/?search=%3Ahas" status="Selectors 4" />

## 简明说明

基于子元素、后续兄弟、关系链选择当前元素。

## 代表语法

```css
:has(> img), :has(+ .error)
```

## 什么时候用

- 当你需要解决「基于子元素、后续兄弟、关系链选择当前元素。」这类问题时，可以优先考虑它。
- 如果状态里包含“草案”“支持有限”“实现推进中”，建议用 `@supports`、特性检测或保守 fallback。
- 如果目标用户浏览器较旧，先从 MDN 与 Can I Use 核对支持矩阵，再决定是否进入生产。
