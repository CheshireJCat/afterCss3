# 生成内容、列表、计数器与内容引用

这个分类包含 9 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [@counter-style](at-counter-style.md) — 自定义有序/无序列表标记体系。
- [counter-set](counter-set.md) — 显式设置计数器值，补足 reset/increment。
- [::marker styling](marker-styling.md) — 列表标记内容和样式控制。
- [content 扩展](content-扩展.md) — 生成内容、替代文本、图像内容。
- [quotes and counters](quotes-and-counters.md) — 语言化引号和层级编号。
- [target-* functions](target-star-functions.md) — 引用目标页码、编号、文本，常用于出版。
- [leaders](leaders.md) — 目录点线引导和页码。
- [running elements](running-elements.md) — 页眉页脚复用文档内容。
- [string-set](string-set.md) — 把内容捕获为页面页眉/目录字符串。
