# CSS-related Web APIs and Houdini Capabilities

This category contains 11 capability entries. Each entry includes a short explanation, MDN entry, Can I Use entry, and a readable demo.

- [CSSOM](cssom.md) - Read, parse, serialize, and manipulate CSS rules.
- [CSSOM View](cssom-view.md) - Query viewports, scrolling, and layout boxes.
- [Geometry interfaces](geometry-interfaces.md) - Use geometric objects, matrices, and coordinate transforms.
- [Resize Observer](resize-observer.md) - Observe element size changes when CSS alone cannot notify JavaScript.
- [Font Loading API](font-loading-api.md) - Control font loading and font set state from scripts.
- [Web Animations API](web-animations-api.md) - Use a JavaScript API that shares timing and effects with the CSS animation model.
- [Custom Highlight API](custom-highlight-api.md) - Register ranges in JavaScript and style them with ::highlight(name).
- [Properties and Values API](properties-and-values-api.md) - Register typed and animatable CSS custom properties at runtime.
- [Typed OM](typed-om.md) - Represent CSS values as typed objects.
- [Paint/Layout/Animation Worklets](paint-layout-animation-worklets.md) - Extend parts of the browser rendering pipeline.
- [View Transition API](view-transition-api.md) - Work with CSS view-transition pseudo-elements from JavaScript.
