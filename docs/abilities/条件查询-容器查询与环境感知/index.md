# 条件查询、容器查询与环境感知

这个分类包含 13 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [Media Queries 4 范围语法](media-queries-4-范围语法.md) — 更接近数学表达的断点写法。
- [输入能力查询](输入能力查询.md) — 根据鼠标/触控/笔等交互能力调整 UI。
- [显示与偏好查询](显示与偏好查询.md) — 响应用户系统偏好和无障碍设置。
- [色彩与动态范围查询](色彩与动态范围查询.md) — 针对 P3/HDR/视频动态范围做渐进增强。
- [运行环境查询](运行环境查询.md) — 判断脚本、刷新能力、溢出方式、PWA 显示模式等。
- [Container size queries](container-size-queries.md) — 基于组件容器尺寸而不是视口响应。
- [Container style queries](container-style-queries.md) — 根据容器已计算样式或 custom property 分支。
- [Container scroll-state queries](container-scroll-state-queries.md) — 根据 sticky/snapped/scrollable 等滚动状态响应。
- [容器单位](容器单位.md) — 直接以查询容器尺寸计算值。
- [特性查询函数](特性查询函数.md) — 检测选择器、声明、字体技术或格式。
- [环境变量](环境变量.md) — 适配刘海屏、折叠屏分段视口、虚拟键盘、标题栏等。
- [自定义媒体查询](自定义媒体查询.md) — 把常用媒体条件命名复用。
- [Anchor/position fallback 查询](anchor-position-fallback-查询.md) — 根据 anchor positioning 选择结果调整样式。
