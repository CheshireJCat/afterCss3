# 背景、边框、图像、遮罩与视觉效果

这个分类包含 15 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [多背景与背景控制](多背景与背景控制.md) — 多个背景层、裁剪、定位和尺寸控制。
- [渐变族](渐变族.md) — 无需图片的线性/径向/锥形渐变。
- [image-set()](image-set.md) — 按分辨率、格式、类型选择资源。
- [cross-fade()/image()](cross-fade-image.md) — CSS 内部图像混合、fallback 和方向性图像。
- [object-fit/object-position](object-fit-object-position.md) — 替代背景图 hack 控制替换元素裁剪。
- [corner-shape](corner-shape.md) — 圆角之外的斜角、凹角、squircle 等角形。
- [border-shape / partial borders](border-shape-partial-borders.md) — 复杂边框路径、部分边框和 box decoration 扩展。
- [box-shadow longhands](box-shadow-longhands.md) — 把阴影拆成可单独动画/覆盖的长属性。
- [masking](masking.md) — Alpha/luminance 遮罩、裁剪路径、复杂形状。
- [basic shapes / shape()](basic-shapes-shape.md) — CSS 原生可参数化形状函数。
- [filters](filters.md) — 元素或背景图层后处理效果。
- [blend modes](blend-modes.md) — 控制图层合成和混合。
- [image-rendering](image-rendering.md) — 像素图/低分辨率资源缩放策略。
- [CSS Image Animation](css-image-animation.md) — 控制动画图像格式播放和首帧等行为。
- [Gap decorations](gap-decorations.md) — 给 grid/flex/multicol gap 绘制线条和装饰。
