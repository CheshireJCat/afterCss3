# 注册自定义属性

<FeatureLinks mdn="https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/At-rules/@property" caniuse="https://caniuse.com/?search=%40property" status="CSS Houdini / 广泛实现中" />

## 简明说明

让自定义属性可动画、可类型检查、可设初始值。

## 代表语法

```css
@property --angle { syntax: '<angle>'; inherits: false; initial-value: 0deg; }
```
