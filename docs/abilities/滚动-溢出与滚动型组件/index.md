# 滚动、溢出与滚动型组件

这个分类包含 10 个能力条目。每个条目都包含简明说明、MDN 入口、Can I Use 入口和可运行/可阅读的 demo。

- [Scroll snap](scroll-snap.md) — 原生吸附滚动、轮播和分页滚动。
- [Scroll behavior](scroll-behavior.md) — 原生平滑滚动。
- [Overscroll behavior](overscroll-behavior.md) — 控制滚动边界链式传播和系统回弹。
- [Scrollbar styling](scrollbar-styling.md) — 滚动条颜色、宽度和布局预留。
- [Scroll anchoring](scroll-anchoring.md) — 内容插入时保持视口位置，减少跳动。
- [Overflow clip](overflow-clip.md) — 不产生滚动容器的裁剪和裁剪边距。
- [line clamp / block ellipsis](line-clamp-block-ellipsis.md) — 多行文本截断和块级省略。
- [CSS carousels](css-carousels.md) — 无需 JS 的滚动按钮、分页点和滚动目标组。
- [Scroll-state queries](scroll-state-queries.md) — 滚动状态驱动样式，常用于 sticky header 阴影等。
- [Spatial navigation](spatial-navigation.md) — 遥控器/键盘方向键空间导航。
